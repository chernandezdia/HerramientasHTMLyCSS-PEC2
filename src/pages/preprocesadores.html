Preprocesadores

Los paquetes de módulos son herramientas que los desarrolladores frontales utilizan para agrupar módulos JavaScript en un solo archivo JavaScript que se puede ejecutar en el navegador.

Un preprocesador te permite escribir CSS en un idioma alternativo y ofrece funciones avanzadas como variables, funciones y la posibilidad de incluir archivos para organizar mejor tu código. Una vez que el código CSS esté escrito en este nuevo idioma, se compilará para volver a transformarse en CSS nativo.
